##  Features

| Feature | Description |
|---|---|
|  **Particle Background** | 120 animated floating particles on canvas |
|  **Glitch Effect** | Name glitches with RGB split every 3 seconds |
|  **Typewriter** | Cycling roles with realistic typing animation |
|  **Custom Cursor** | Glowing dot + trailing ring that follows mouse |
|  **Scroll Progress Bar** | Neon gradient bar tracks reading progress |
|  **Spring Animations** | Every card springs in on scroll with framer-motion |
|  **Fully Responsive** | Mobile hamburger menu, fluid typography |
|  **Resume Link** | One-click access to Google Drive resume |

##  Run Locally

```bash
# Clone the repo
git clone https://github.com/DakshMehrotra/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open `http://localhost:5173` in your browser.
